Sun Devil Bank Account Management System

PROJECT DESCRIPTION

This project involves the development of a dynamic hash table-based system for managing bank account records at Sun Devil Bank. The system is designed to handle a maximum of 800 accounts at any given time. The implementation of the hash table is done in two distinct methods: Hashing with chaining for collision handling and Hashing with open addressing using linear probing.

Key Features:

  • Efficient Collision Handling
  • Dynamic Account Record Management
  • Scalable Design for Account Handling
  • User-Friendly Interface for Account Operations
